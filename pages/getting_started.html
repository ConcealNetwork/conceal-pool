Algorithm

Cryptonight Conceal

Username

The username is your wallet address. Any pool earnings will then be sent to that address once your miner reaches the minimum payment threshold. You can see your miners progress in the workers page once you enter your address. You can also mine directly to an Exchange address, although we recommend mining to your own wallet, and then transfer to an Exchange.


Mining to a Wallet: Address

Mining to an Exchange: Address+PaymentID

Set Static Difficulty: Address.difficulty

Below is a list of mining software that supports Cryptonight Conceal.

XMRig

(forked version of the XMRig CPU Miner) "pool_list": [
{
"pool_address": "us.pool.conceal.network:5555",
"wallet_address": "YOUR_WALLET_ADDRESS",
"rig_id": "YOUR_WORKER_NAME",
"pool_password": "x",
"use_nicehash": false,
"use_tls": false, /* Set to true if you are using an SSL port */
"tls_fingerprint": "",
"pool_weight": 1
},
],
"currency": "cryptonight_conceal",

XMR-Stak

(CPU+GPU Miner) "pools": [
{
"url": "us.pool.conceal.network:5555",
"user": "YOUR_WALLET_ADDRESS",
"pass": "YOUR_WORKER_NAME",
"keepalive": true,
"nicehash": false,
"variant": "conceal"
}
],

SRBMiner

(AMD GPU Miner)

Double click on 'guided-setup.bat', and the program will guide you thru the setup process.

XMRigCC

(XMRigCC is a fork of XMRig which adds the ability to remote control your XMRig instances) xmrigDaemon.exe -o us.pool.conceal.network:5555 -u YOUR_WALLET_ADDRESS -p YOUR_WORKER_NAME -a cryptonight --variant conceal -k
